How to play

  1. The Admin picks the leagues (NBA, MLB, NFL, NHL) and whether players can be current, past, or both.
  2. The first player names any eligible athlete — say, Michael Jordan.
  3. Take the first letter of that last name (Jordan → J). The next player must name an athlete whose first name starts with J — like John Stockton.
  4. Then Stockton → S, so the next first name starts with S, and so on.
  5. No repeats. You have 30 seconds per turn. Run out of time and you're eliminated. Last player standing wins.
