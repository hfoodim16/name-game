How to play

  1. The Admin picks the leagues (NBA, MLB, NFL, NHL), whether players can be current, past, or both, and the turn timer.
  2. The first player names any eligible athlete — say, Michael Jordan.
  3. Take the first letter of that last name (Jordan → J). The next player names an athlete whose first name starts with J — like John Stockton.
  4. Then Stockton → S, so the next first name starts with S, and so on.
  5. No repeats. Run out of time (or tap Stuck) and you're eliminated. Last player standing wins.

Timer, Pause & Challenge

  • Turn timer — set it to 15–90 seconds, or Off for relaxed pass-and-play with no clock. With the timer off, tap Stuck when you can't think of anyone.
  • Pause ⏸ — freezes the clock for a bathroom break, snack run, or argument. Resume right where you left off.
  • Challenge 🚩 — the app checks names automatically, but it isn't perfect. If you think an answer was wrongly rejected (a real player it didn't know) or shouldn't have counted (wrong league/era, or against your house rules), hit Challenge. The game pauses so everyone can discuss, then you decide together whether it counts — Allow it, or throw it out and redo the turn.
